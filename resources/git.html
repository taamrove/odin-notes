git init To the top

- git init is used to initialize a Git project locally before connecting it to a Github repository, so all Git functions can be used while beeing offline for example.

git add To the top

- check status of the Git in the current project, see what files are edited, not added or commited, and which branch you are on.

- add a single file, multiple files joined by space, or all files using the "add ."

git commit To the top

- makes a snapshot of the added files at the moment of commit.

- need the -m flag for commenting on the features of the commit.

git push To the top

- pushes all commits to Github.

- can be used as a simple "git push", automatically pushing origin to main

- can also be specified by "git push COMMIT BRANCH"

git checkout To the top

- makes you able to have a look into previous commits.
