cd

- cd command is used to navigate.

- "cd" takes you to root folder.

- "cd .." navigates a folder back.

- "cd ../PATH" or "cd ../../PATH/PATH" navigates X times backwards and then into folders.

- "cd PATH" or "cd PATH/PATH" navigates into the folders.

- "cd ~/PATH" the ~ is a placeholder for the working directory.
